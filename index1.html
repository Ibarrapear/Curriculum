Que es HTML?

HTML es el acrónimo de HyperText Markup Language, que en español significa Lenguaje de Marcado de Hipertexto,
es un lenguaje de programación utilizado para crear páginas web y aplicaciones para la web.

Como es la estructura de HTML

El HTML se compone de una serie de etiquetas que se utilizan para indicar cómo debe ser presentado el contenido de una página web.
Cada etiqueta tiene una función específica, como crear un encabezado, un párrafo, una lista o un enlace, estas etiquetas se escriben
dentro de un archivo de texto, que se guarda con la extensión .html. Cuando se abre el archivo con un navegador web,
el navegador lee el código HTML y muestra la página web según las instrucciones que se han indicado.

Como se relaciona ?

El HTML es uno de los lenguajes de programación más importantes en la creación de páginas web.
Junto con otros lenguajes como el CSS y el JavaScript, se utilizan para diseñar y crear sitios
web interactivos y atractivos.
